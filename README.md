## 选择器
作用：选择页面上的某一个或某一类

1.基本选择器
- 标签选择器 选择一类标签 `标签{}`
- 类选择器  选择class属性一致的标签 `.类名{}`
- id选择器  全局唯一  `#id名{}`

*顺序* 

**id选择器>类选择器>标签选择器**，无就近原则


2.高级选择器
- 层次选择器
1. 后代选择器
2. 子选择器
3. 相邻兄弟选择器
